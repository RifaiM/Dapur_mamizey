Format Order via WhatsApp :

  1. Nama
  2. Alamat Lengkap
  3. Nama Produk Pesanan
  4. Jumlah Pesanan
  5. Cara Pembayaran (cash/transfer)

Notes : Cantumkan juga keinginan waktu pengiriman.

Format Order via DM Instagram :

  1. Nama
  2. Alamat Lengkap
  3. Nama Produk Pesanan
  4. Jumlah Pesanan
  5. Cara Pembayaran (cash/transfer)

Notes : Cantumkan juga keinginan waktu pengiriman. Demi kenyamanan, disarankan untuk chat Penjual terlebih dahulu melalui WhatsApp. Informasi kontak silahkan klik disini.

Format Order via Email :

  1. Subject email : Pemesanan
  2. Nama
  3. Alamat Lengkap
  4. Nama Produk Pesanan
  5. Jumlah Pesanan
  6. Cara Pembayaran (cash/transfer)

Notes : Cantumkan juga keinginan waktu pengiriman. Demi kenyamanan, disarankan untuk chat Penjual terlebih dahulu melalui WhatsApp. Informasi kontak silahkan klik disini.

Cara Pembayaran :

  1. Cash
    • Menerima Cash on Delivery (COD)
    • Untuk kemudahan bersama, mohon siapkan uang pas
  2. Bank Transfer
    • Chat Penjual terlebih dahulu (memastikan produk tersedia)
    • Setelah konfirmasi produk tersedia, silahkan transfer ke rekening BCA a/n Sudrajat (xxx xxx)
    • Kirimkan bukti transfer via Chat WhatsApp ke Penjual
    • Produk akan kami kirimkan sesuai format order (sesuai juga dengan keinginan waktu pengiriman)

Untuk pembayaran via Bank Transfer, mohon kirimkan foto bukti struk asli atau bukti screenshot jika transfer melalui m-banking.
